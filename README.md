## About Skillable

Skillable.API, skillable that can create, read, update and delete your skills.

## Installation 

```sh
composer install
```

```sh
composer update
```

```sh
php artisan key:generate
```

```sh
create database and name it as 'skillsdb'
```

```sh
php artisan migrate:fresh
```

```sh
php artisan serve
```
